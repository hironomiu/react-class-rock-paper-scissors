# react-rock-paper-scissors
じゃんけんゲーム

## 環境
Redux、React Redux、Redux DevTools、redux-persist
```
$ npm install --save redux
$ npm install --save react-redux
$ npm install --save-dev redux-devtools
$ npm install --save redux-persist
```

## 実行
```
$ npm start
```

## ビルド
```
$ npm run build
```

## PORT設定
`.env`にて3001を設定
